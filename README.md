# Grade-Calculator
A simple and beginner-friendly Grade Calculator built using Python.
This program takes marks from the user, calculates the total, percentage, and assigns a grade based on standard criteria.
## Features
1.Takes marks for multiple subjects
2.Calculates:
   Total marks
   Average / Percentage
   Final Grade
3.Handles invalid input
4.Beginner-friendly Python code
5.Easy to extend for more subjects
## How it works
1.Asks the user to enter marks for each subject
2.Adds all marks
3.Finds percentage
4.Checks grade based on percentage using if-elif statements
## Grade Criteria
Percentage	Grade
≥ 90	        A+
≥ 80        	A
≥ 70        	B
≥ 60        	B+
≥ 50       	  C
≥ 40	        C+
≥ 30          D
≥ 20          F
## How to Run
1.Download or clone this repository
2.Open the .py file in VS Code / PyCharm / Google Colab
3.Run the program
4.Enter marks when asked
